# 简单介绍

一个纯Python编写的web框架

# 安装

一般情况

```
pip install Django
```

非一般情况，这里用Anaconda

```
conda install Django
```

安装好后版本验证

```
python -m diango --version
```

# 创建项目

简单版（新手版本）

先cd到一个想要存放代码的地方

然后：

```
django-admin startproject xxxxx
```

这样就可以得到一个项目了：
